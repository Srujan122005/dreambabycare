Place your feeding videos here to make them available in the Tips -> Video Guides -> Feeding tab.

Recommended filename:
  feeding_breast_or_bottle.mp4

Windows copy command (from Downloads):
  copy "%USERPROFILE%\Downloads\invideo-ai-1080 Breast or Bottle_ A Gentle 1_50 Guide 2026-01-10.mp4" "%~dp0..\..\static\videos\feeding\feeding_breast_or_bottle.mp4"

Or use File Explorer to drag the file into this folder.

Notes:
- Videos will be served via the `/protected_video/videos/feeding/<filename>` endpoint and require a valid subscription to view.
- After copying, reload the `/tips` page while logged in to see the video listed under the Feeding tab (subscribers only).
